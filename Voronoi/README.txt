Aplicação do diagrama de Voronoi:

Em visão computacional, o  diagrama de voronoi é uma ferramenta valiosa para o processamento de imagens, especificamente,
para a segmentação de imagens e indentificação de pontos de interesse para análise. Ao identificar pontos de alto contraste
e aplicando o diagrama de voronoi, é possível identificar seções da imagem onde estes picos são prevalentes. 

Como um exemplo concreto, identificação de células em fotos de microscópio usando o diagrama de voronoi, que é uma maneira
muito menos custosa quando comparado com a alternativa de utilizar machine learning para essa Aplicação
